### README для взаимодействия с приложением `TruckStacker`

#### Входные данные

При запуске программы, пользователю необходимо будет ввести следующие данные:

1. **Имя файла с данными о грузовиках**:
  - Введите полное имя файла с расширением `.json`, содержащего информацию о грузовиках.
  - Пример ввода: `trucks.json`

2. **Имя файла с данными о коробках**:
  - Введите полное имя файла с расширением `.txt`, содержащего информацию о коробках.
  - Пример ввода: `boxes.txt`

3. **Тип алгоритма укладки**:
  - Введите тип алгоритма для укладки коробок. Доступные варианты:
    - `balanced`
    - `burke`
    - `dummy`
  - Пример ввода: `balanced`

4. **Максимальное количество используемых грузовиков**:
  - Введите целое число, определяющее максимальное количество грузовиков, которое может быть использовано для укладки.
  - Пример ввода: `3`

#### Пример интерактивного ввода

```
Enter 'exit' to stop or press Enter to continue:
[пустая строка] (нажмите Enter)

Write filename with trucks data (.json):
trucks.json

Write filename with boxes data (.txt):
boxes.txt

Write type algorithm (balanced, burke, dummy):
balanced

Write max size of truck usage:
3
```

#### Завершение работы

- Чтобы завершить выполнение программы, введите `exit` на первом этапе интерактивного ввода.

#### Примечания

- Убедитесь, что файлы с данными о грузовиках и коробках существуют и корректно отформатированы по указанным спецификациям (JSON и TXT соответственно).
- После ввода всех данных программа выполнит операцию укладки и выведет результат, использующий указанный алгоритм.
